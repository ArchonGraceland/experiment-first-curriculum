# Experiment-First Science Curriculum

A high school science curriculum (Physics, Chemistry, Biology, Earth & Space Science) that teaches through experiments instead of definitions. Designed for a 16-year-old with ADHD.

**Project hub:** [https://archongraceland.github.io/experiment-first-curriculum/](https://archongraceland.github.io/experiment-first-curriculum/)

## Structure

```
├── index.html                          ← Project hub (GitHub Pages site)
├── files/                              ← Current markdown documents
│   ├── curriculum_spec.md              ← Curriculum design specification
│   ├── assessment_spec.md              ← Assessment system specification
│   ├── module_c2.md                    ← Module C2 student document
│   ├── c2_instructor_guide.md          ← Module C2 instructor guide
│   └── c2_lab_notebook.md              ← Module C2 lab notebook worksheet
├── experiment_first_curriculum_spec.docx  ← Original spec (Word)
├── assessment_spec.docx                   ← Original assessment spec (Word)
├── curriculum_equipment_budget.xlsx        ← $5K equipment budget
├── experiment_guide.md                    ← Early experiment notes
├── experiment_upgrade_notes.md            ← Equipment upgrade notes
└── research_report_tool.jsx               ← Early report tool prototype
```

## Curriculum Overview

- **Core:** 48 modules (~24 weeks) across 4 NGSS domains
- **Advanced:** 14 modules (~7 weeks) for deeper exploration
- **Total:** 62 modules, ~31 weeks, ~2 hours/week

## Status

**Proof of concept complete.** Module C2 (Fruit Battery) is fully built with student module, instructor guide, lab notebook, and a deployed research report tool with AI feedback.

See the [project hub](https://archongraceland.github.io/experiment-first-curriculum/) for full status and all documents.
